DBC-Forum
=================
This was the first DBC group project where we used Rails and RSpec testing. We created a simple Q/A site with upvote/downvote and login capabilities.

To Run
-----------------
```ruby
bundle install
rake db:create
rake db:migrate
rails server
```
(to run test suite)
```ruby
rake spec 
```

Dependencies
-----------------
- RSpec for testing
- PostgresSQL for database
- Rails Version: 4.0.0
- Ruby Version: 1.9.3
